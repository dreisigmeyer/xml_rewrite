October 23rd, 2001

U.S. Patent and Trademark Office

Information Products Division

Data Dissemination Grant Red Book Announcements

The following changes have been implemented in Grant Red Book version 2.5 xml DTD (zipped). Changes since the September 25th notice are highlighted in red.

  • Modified PUBLIC identifier to: "-//USPTO//DTD ST.32 US PATENT GRANT V2.5 2002-01-01//EN"
  • Modified element CLREF from EMPTY to PDAT.
  • Deleted B863 & B864 and replaced with optional B864US for the 371 (c)(1)(2)(4) PCT date.
  • B645 is now used for reexaminations requested by the applicant, assignee, or commissioner.
  • Added element B646US for reexamination requested by a 3rd party.
  • Sequence listing elements S-2-IX-D content model replaced with (STEXT+).
  • Removed attribute FILE from element PATDOC.
  • SEQLST-US content model removed the EMI* reference.
  • Kind code changes effective 2001-01-02 to accommodate pre-grant publication status (B130) required a comment change in the dtd.
  • Modified the following elements to include an ALT attribute for compliance to section 508 of the Rehabilitation Act Amendments of 1998. The element will include the alt attribute specifying the default value.
    • CHEMCDX with default value "chemistry chemdraw file"
    • CHEMMOL with default value "chemistry mol file"
    • CUSTOM-CHARACTER with default value "custom character"
    • EMI with default value "embedded image"
    • MATHEMATICA with default value "mathematica file"
    • RTI with default value "replaced text with image"
  • CHEM-US and MATH-US content mode changed from EMI* to EMI?. A single image file is provided. The file however may be a multi-page tiff image.
  • Changed the attribute sequence of CUSTOM-CHARACTER to conform to the sequence used by the other file link elements.
  • Modified sequence listing element S-2-VI-A (organism) to be optional.
  • Re-added attributes HE, WI, LX, and LY to elements CUSTOM-CHARACTER, RTI, and EMI. They had been removed in version V2.3 with the assumption that the information could be extracted from the TIFF header. Attributes LX and LY as well as element RTI are reserved for future use. Release 2.5 data will not include attributes LX and LY or element RTI.
  • Added element B690US (prior publication) to related applications. Multiple B690US elements may exist.
  • Added optional attribute LANG to element DOC. This change is reserved for future use. Release 2.5 data will not include this change.

The following changes have been implemented in Grant Red Book version 2.5 xml DTD (zipped) to convert the dtd from SGML to xml:

  • Modified the contact information and added the xml processing instruction.
  • Removed all element minimization parameters (- -,o -,- o,o o)
  • Modified the dtd to utilize the patent application publication entity and notation references, as well as the cals-xml and original mathml DTDs.
  • Element S-2 attribute SEQ-NO changed from NUMBER to CDATA.
  • Changed element references of MATH to math to comply with the mathml dtd. Elements F and MATHML were impacted. In SGML elements are case insensitive, and in the production data the lower case math element tag mapped to the MATH element. However in xml the elements and attributes are case sensitive. Since the data was already lower case, the DTD was modified to conform to the data.
  • Changed element references of TABLE to table to comply with the CALSxml.dtd. Elements TABLE-CALS and SEQLST-US were impacted.
  • The DTD was modified to reference the local entity file ST32-US-Grant-025xml.ent which contains the external file entity declarations. The DTD no longer needs to be modified for entity file path changes, and no longer has any hard-coded path information.

The Grant Red Book version 2.5 xml DTD (zipped) is a final release for initial production use in January 2002. If you have any questions, or issues with these changes, please respond via e-mail to ed.johnson@uspto.gov.
